# an-lisismundial2026
Informe analítico interactivo del Mundial 2026 con datos de los 48 equipos clasificados, organizados por confederación y grupos. Incluye estadísticas reales de los últimos 20 partidos: goles, corners, tarjetas, posesión, forma reciente, probabilidades estimadas de resultado y picks arriesgados por equipo. Solo para entretenimiento.
